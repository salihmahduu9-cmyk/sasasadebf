أداة ويب قوية لفك تشفير وإلغاء الـ Obfuscation من الأكواس البرمجية

## ✨ المميزات

- ✅ فك تشفير **Lua Scripts** (MoonVeil Obfuscator)
- ✅ فك تشفير **JavaScript**
- ✅ دعم **Python**
- ✅ تجميل وتنسيق الأكواد
- ✅ إزالة الأسماء المشفرة وإعادة تسميتها
- ✅ فك تشفير النصوص المرمزة (Hex, Unicode)
- ✅ يعمل بدون الحاجة لإرسال البيانات للخادم (Client-side)
- ✅ واجهة سهلة الاستخدام
- ✅ دعم النسخ واللصق السريع

## 🚀 الاستخدام

### الطريقة الأولى: استخدام الموقع مباشرة
```
1. افتح الموقع في المتصفح
2. اختر لغة البرمجة (Lua, JavaScript, Python)
3. الصق الكود المشفر في صندوق Input
```
